## 📚 精品课程更新说明

### 课程更新情况

目前网站"精品课程"中的课程更新情况如下：

- **热门课程**：更新视频较多，部分特别热门的课程已经全部更新完成
- **已完结课程**：已完成全部视频更新的课程，会在标题中标注 **"已完结"** 标识
- **冷门课程**：部分冷门课程可能仅更新了第一节视频

### 遇到"资源不存在"提示

如果您在使用配音插件时遇到 **"资源不存在"** 的提示，说明该视频尚未完成配音处理。

**解决方法：**

1. 在用户群中通知我们，我们会尽快安排更新
2. 等待产品后续版本上线，届时将支持任意视频的配音功能，使用会更加便捷

:::tip 温馨提示
我们正在积极完善课程库，优先更新热门课程。如果您有特别想学习的课程，欢迎在用户群中反馈，我们会根据需求调整更新优先级。
:::

## 🔗 课程跳转说明

### 跳转方式

点击"精品课程"中的课程封面，大多数课程会直接跳转到对应的 **YouTube 播放列表（Playlist）**。

### 特殊课程处理

部分课程（如 **CS61A**、**CS61B** 等）的视频分散在 YouTube 的不同播放列表中，目前会直接跳转到**课程官网**。

**使用方式：**

1. 在课程官网中找到 **Video**、**Recording** 等链接
2. 点击链接跳转到对应的 YouTube 视频地址
3. 在 YouTube 视频页面使用配音插件进行观看

:::tip 温馨提示
跳转到课程官网的设计，有助于您更好地了解课程的整体结构、学习资料和配套资源。建议您在学习视频内容的同时，也熟悉课程官网上的材料（如课程大纲、作业、阅读材料等），这样可以获得更完整的学习体验。
:::

## 🎁 邀请返利活动说明

### 活动规则

关于邀请返利活动，需要特别说明以下几点：

| 情况 | 说明 |
|------|------|
| ✅ **可获得优惠券** | 邀请到**未订阅过的新用户**，且新用户**首次订阅**后 |
| ❌ **不显示优惠券** | 邀请的是**已经订阅过的用户**（即使"邀请好友"中有记录） |

### 注意事项

- 邀请记录会在"邀请好友"中正常显示
- 但只有满足上述条件时，才会在"已获得优惠券"中显示
- 请确保邀请的是从未订阅过的新用户

:::warning 重要
只有新用户首次订阅后，您才能获得相应的优惠券奖励。已订阅过的用户不会触发返利。
:::
