# 常见问题

在这里您可以找到使用 IvyGo 艾维果过程中常见问题的解答。

## 🔧 安装问题

### Q1: 插件安装失败怎么办？

**解决方法：**

请检查以下几点：
- ✅ 浏览器版本是否在 142 以上
- ✅ 网络连接是否正常
- ✅ 是否有足够的存储空间
- ✅ 尝试重启浏览器后再安装

### Q2: 找不到插件图标？

**解决方法：**

插件图标可能被隐藏了，按以下步骤操作：

1. 点击浏览器地址栏右侧的**拼图图标** 🧩
2. 在列表中找到 **IvyGo 艾维果**
3. 点击旁边的**图钉图标** 📌 固定到工具栏

![固定插件图标](/img/plugin-guide/CleanShot%202025-12-02%20at%2017.35.21@2x.png)

### Q3: 开发者模式开关在哪里？

**Chrome 浏览器：**
- 打开 `chrome://extensions/`
- 开关在页面**右上角**

**Edge 浏览器：**
- 打开 `edge://extensions/`
- 开关在页面**左下角**

## 🎬 使用问题

### Q4: 插件不工作怎么办？

**解决方法：**

请尝试以下步骤：

1. **刷新当前网页** - 按 `F5` 或 `Ctrl+R` (Mac: `Cmd+R`)
2. **重启浏览器** - 完全关闭后重新打开
3. **检查插件状态** - 确认插件没有被禁用
4. **重新登录** - 退出账号后重新登录

### Q5: 翻译按钮点击没反应？

**解决方法：**

- 确保您已登录账号
- 检查网络连接是否正常
- 等待页面完全加载后再点击
- 尝试刷新页面后重试

### Q6: 翻译时间过长？

**说明：**

翻译时间取决于：
- 视频长度
- 网络速度
- 服务器负载

**建议：**
- 耐心等待，直到按钮变为**绿色**
- 避免在翻译过程中刷新页面
- 确保网络连接稳定

### Q7: 翻译后没有声音？

**解决方法：**

1. 检查电脑/浏览器是否静音
2. 检查 YouTube 播放器音量设置
3. 尝试刷新页面后重新播放

## 🔐 账号问题

### Q8: 如何注册账号？

首次使用时，在插件登录界面点击**注册**，按照提示完成注册流程。

### Q9: 忘记密码怎么办？

在登录界面点击**忘记密码**，通过注册邮箱/手机号重置密码。

### Q10: 可以在多个设备使用吗？

是的，您可以在多个设备上登录同一账号使用。

## 🌐 兼容性问题

### Q11: 支持哪些浏览器？

目前支持：
- ✅ Google Chrome（推荐）
- ✅ Microsoft Edge

暂不支持：
- ❌ Firefox
- ❌ Safari

### Q12: 支持哪些视频网站？

目前仅支持 **YouTube** 视频翻译。
