# 欢迎使用

**IvyGo 艾维果** 是一款强大的浏览器插件，让您能够轻松将 YouTube 视频内容翻译成中文语音，保持音色语调，流畅自然，打破语言障碍，畅享全球优质课程。

## ✨ 核心功能
- 🎬 **视频配音** - 一键将 YouTube 视频从英文语音切换成中文语音，保持音色语调，流畅自然
- 🎯 **双语字幕** - 一键将 YouTube 视频显示高质量中英文字幕
- 📚 **课程学习** - 内置课程列表，快速访问优质学习资源，帮助您减少信息盲区
- 🔄 **实时响应** - 快速处理，翻译完成后立即可播放

## 🌟 产品亮点

### 突破语言障碍
无需精通英语，也能学习全球顶尖的教育内容。艾维果让优质知识触手可及。

### 简单易用
只需几步简单操作，即可完成视频翻译。无需复杂设置，开箱即用。

### 学习无界限
访问来自世界各地的课程和教程，让学习不再受语言限制。

## 🚀 快速开始

准备好开始使用艾维果了吗？按照以下步骤快速上手：

1. **[安装插件](/docs/installation)** - 在浏览器中安装艾维果插件
2. **[登录账号](/docs/usage)** - 注册并登录您的账号
3. **[开始翻译](/docs/usage#-翻译视频)** - 选择视频，一键翻译
