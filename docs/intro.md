# 欢迎使用

**IvyGo 艾维果** 是一款强大的高质量浏览器 AI 配音插件，能够轻松将 YouTube 英文视频内容翻译成中文语音，同时保持原说话人的音色和语调，使配音听起来流畅自然。通过使用母语接收信息，基本消除了语言壁垒，大幅提高学习效率。

## ✨ 核心功能
- 🎬 **视频配音** 一键将 YouTube 视频从英文语音切换成高质量的中文语音，同时保持说话人的音色和语调，听起来流畅自然。使用母语学习，效率更高。
- 🎯 **双语字幕** 支持显示高质量的中英文字幕，可使用鼠标拖拽调整字幕位置。您也可以仅显示双语字幕而不使用中文配音，根据需求灵活调整。
- 📚 **课程学习** 内置课程列表，帮助您快速找到优质学习资源，减少信息盲区。目前仅支持课程列表中的视频配音，后续将支持 YouTube 网站上任意视频的配音功能（正在开发中）。如果您想观看的视频不在支持列表中，可通过企业微信群联系我们。
- 🔄 **实时响应** 配音秒级切换，无需等待，操作流畅。

## 🌟 产品亮点

### 原声质感
通过持续的技术优化，在中文配音中保留了说话人的音色和语调，听起来就像说话人学会了中文，流畅自然，没有违和感。

### 洗经伐髓
世界名校的老师讲课往往深入浅出，但课程难度高、挑战大，学习过程需要改变认知、磨练大脑，如同洗经伐髓般痛苦，很多人难以坚持。消除语言壁垒后，完成挑战的概率将大幅提升。

### 成神之路
跟着大师学习，才有可能成为大师。消除语言壁垒后，能够更多更深入地吸收来自大师的思想精华，开启属于自己的成神之路。

### 简单易用
只需几步简单操作，即可完成视频翻译。无需复杂设置，开箱即用。

### 学习无界限
访问来自世界各地的课程和教程，让学习不再受语言限制。
